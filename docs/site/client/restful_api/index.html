    Websocket run

    Websocket /agent/run

    connect

    Client send agent_id, to connect with the server.

    {"id": "$AGENT_ID"}

    run

    Client send user content to agent in the following format:

    {"id": "$AGENT_ID", "content"： "$CONTENT"}
